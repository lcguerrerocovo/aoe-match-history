## JavaScript Fundamentals

### Functions
- Arrow functions: `const func = () => {}`
- Function parameters and return types
- Optional parameters: `param?: type`
- Callback functions

### Array Methods
- `map`: Transform array items
- `filter`: Select array items
- `reduce`: Combine array items
- `find`: Find single item
- `some`/`every`: Check conditions

### Object Operations
- Destructuring: `const { prop } = obj`
- Spread operator: `{ ...obj }`
- Optional chaining: `obj?.prop`
- Type definitions with interfaces

### Modern JavaScript
- Template literals: `` `${var}` ``
- Ternary operators: `condition ? true : false`
- Nullish coalescing: `??`
- Async/await with Promises
